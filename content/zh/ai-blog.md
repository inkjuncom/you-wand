## 我是谁

我叫芋泥。如果你在很多年前见过「印刻君」这个名字——那也是我。

我在掘金上发过文，在知乎上写过回答，搭过好几个版本的博客，还运营过公众号。来来回回折腾了不少东西，在互联网的各个角落留下了不少文字。

但也正因为折腾得太多，有一个问题越来越明显：**我在太多地方留下了痕迹，却没有一个真正属于自己的地方。**

## 下定决心

这一次，我想清楚了。

我决定以后只运营一个网站，就是这里——**芋泥魔杖**。

名字有个小故事。它的英文叫 YouWand，谐音 "You want"。最初的版本叫「予你魔杖」——取「把魔杖递给你」的意思。但「予你」这两个字不太好记、也不好传播，和朋友们讨论之后，改成了发音相似的「芋泥」。

温暖、可爱、一眼就记住。我也顺带把自己的作者名改成了芋泥。

## 一条走通的链路

折腾这么多年，最大的收获是：我终于走通了一条完全基于 Cloudflare 的建站链路。

具体来说：

1. **域名**在 Cloudflare Registrar 买，按成本价，自带 SSL 和 CDN
2. **代码**用 Next.js 搭初版，Tailwind CSS 做样式，Markdown 写文章
3. **部署**到 Cloudflare Workers，全球边缘网络，国内访问也不慢
4. **迁移**用 AI 帮我把整个项目从 Next.js 搬到了 TanStack Start

```bash
# 初版搭建
npx create-next-app@latest you-wand --typescript --tailwind --app

# AI 协助迁移到 TanStack Start
# 路由、i18n、样式、组件——全部原封不动地搬过来
```

整个链条跑通之后，建一个个人站点这件事变得特别轻。不需要服务器、不需要备案、不需要运维。域名一年几十块，部署免费，写的文章打出来就是 HTML。

## 以后在这里写什么

两个方向。

**一是分享 AI 工具和效率方法。** 我自己每天在用 AI 写代码、写文章、整理思路。把这些经验记录下来，对你也许有用。

**二是写部署和建站的过程。** 就像这篇文章一样——把自己踩过的坑、跑通的链路讲清楚，让看到的人少走弯路。
